A cloud-native AI-powered crypto analytics platform built with AWS Bedrock, Next.js, and Anthropic Claude.

Problem Statement

Cryptocurrency markets are fast, volatile, and complex. Most beginner investors struggle to understand price movements, technical indicators, market sentiment, trend direction, and risk levels. Existing tools are either too technical for newcomers or too shallow to be actionable.

CryptoSight AI was built to bridge this gap — providing an AI-powered financial analyst that explains what's happening in the market in plain English, making crypto accessible to everyone.

Solution & Key Features

Real-time crypto price retrieval for BTC, ETH, SOL and more

Technical indicator analysis — RSI, EMA, SMA, and volatility

Natural-language trend explanations powered by Amazon Bedrock

Q&A chatbot for investment education via Anthropic Claude

Basic portfolio & watchlist support

Educational explainers designed for beginners

Target Users

Individual Investor

Monitor top cryptos, view charts, ask AI questions to make informed decisions.

Beginner / Student

Ask educational questions and build crypto knowledge safely.

Financial Analyst

Request market analysis and compare assets for data-driven recommendations.

Day Trader

Switch between crypto charts quickly and track 24h changes for intraday trades.
